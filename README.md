# Platypus
Heavy reconnaissance robot simulator for search-and-rescue missions. Inspired by the real Portuguese robot RAPOSA and the vehicle physics of MudRunner. Mud. Stairs. Rubble piles. Slow, tense control. One wrong angle — and you're stuck forever. We're working to make every single stair climb feel like a small, hard-earned victory.


Симулятор тяжёлого разведывательного робота для поисково-спасательных миссий. Вдохновлён реальным португальским роботом RAPOSA и физикой MudRunner.
Грязь. Лестницы. Завалы. Медленное, напряжённое управление. Только один неверный угол — и ты застрял навсегда.
Работаем над тем, чтобы каждый подъём по лестнице ощущался как маленькая победа.
